Floats

We have already seen in the last section that often it is useful to enclose a tabular environment in a table environment. The table environment is an example of a float. Floats are basically blocks of content that "float" around the page in the sense that LaTeX chooses where to place them based on certain algorithms. In general, it is a good idea to let LaTeX place the float where it wants, at least at first, when you have the entire document written you can then go back and change it if you wish. If you use the float package, one can override the LaTeX placement with the following syntax and specifier.

The code below right gives an example of a typical float environment with special placement. Note the float package is not necessary if you wish to let LaTeX place the figure for you. Here the specifier is H. The following table lists some of the possible specifiers (you can of course define your own).

Specifier Action
H Place exactly at spot in source text
h Place approximately at spot in source test
t Place at top of page
b Place at bottom of page
p Place on page for floats only
! Override internal LaTeX parameters for determining float position 
\documentclass{article}
\usepackage{float}
\begin{document}
\begin{figure}[H]
%figure content goes here
%Can be:
%graphic
%table
%tikz drawing
%etc.
\end{figure}
\end{document}
