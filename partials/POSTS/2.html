Delta Function Potential June 21, 2015 The Delta function potential is a great example problem for one-dimensional quantum mechanics. The solution shows off many of the tricks associated with solving these problems and it has features such as a bound state which can be harder to understand in more complicated contexts. Before stating the problem, a quick refresher on the Dirac Delta functions. While not technically a *function*, it is a mathematica object which is very useful for many physical reasons. So we define it as the functoin where $\delta(x) =0$ unless $x=0$ in which case $\delta(0) = +\infty$ and $$\int\_{-\infty}^\infty \delta(x)\,dx = 1.$$ There are are many other useful properties, but this is all we will need for this problem. ### Problem Statement Find the wavefunctions and energies of a 1-D particle of mass m in the potential \begin{equation} V(x) = V\_0 a \delta(x), \end{equation} ### Solution So we want to solve the Schrodinger equation \begin{equation} -\frac{\hbar^2}{2m}\frac{d^2}{dx^2}\psi(x) + V(x)\psi(x) = E\psi(x), \end{equation} where $-\infty\lt x\lt \infty$. So our general idea will be to solve the equation for $x \lt 0$ and for $x \gt 0$ and join at the origin. The free particle solutions can be written as \begin{equation} \psi\_- = A \sin(kx) + B\cos(kx) \end{equation} \begin{equation} \psi\_+ = C \sin(kx)+ D\cos(kx), \end{equation} where $k$ is not necessarily real and defined by \begin{equation} \frac{\hbar^2 k^2}{2m} = E. \end{equation} Our two general boundary conditions are given by the continuity of the wavefunction at $x=0$ and if we integrate both sides of the Schrodinger equation from $-\epsilon $ to $\epsilon$ where $\epsilon \to 0$, we must retain equality. More explicitly, we have that \begin{equation}\psi\_-(0)=\psi\_+(0)\end{equation}, which implies that \begin{equation} -\frac{\hbar^2}{2m}\int\_{-\epsilon}^\epsilon \psi''(x)\, dx + \int\_{-\epsilon}^\epsilon V\_0 a \delta(x) \psi(x)\,dx = \int\_{-\epsilon}^\epsilon E\psi(x), \end{equation} and we can simplify the equation by noting that $\psi$ is continuous, thus $\int\_{-\epsilon}^\epsilon E\psi(x)=0$ and by using the definition of the delta function, we get for the second condition \begin{equation} -\frac{\hbar^2}{2m}[\psi'(x)]\_{-\epsilon}^\epsilon + V\_0 a \psi(0) = 0, \end{equation} which leads to the two conditions \begin{align} B &=D\\ \label{1ddelta} \frac{\hbar^2k}{2m}[C-A]&=V\_0aB \end{align} Now we must consider the cases of a bound state and a scattering state. #### Bound State For this state, we require the wavefunction be normalizable, so we require that \begin{equation} \lim\_{x\to \pm \infty}\psi(x) = 0. \end{equation} So in order to have this, we know $k$ must be imaginary, so we define $\kappa = ik$, which gives us that \begin{equation} \psi\_- = A\left(\frac{e^{-\kappa x}-e^{\kappa x}}{2i}\right) + B\left(\frac{e^{-\kappa x}+e^{\kappa x}}{2}\right), \end{equation} and if we require this to be zero at $-\infty$, we get that $A=-iB$ by noting that the term pre-multiplying the divergent exponential, must be zero. Similar considerations for $\psi\_+$ gives that $C=iD$. Since we know that \begin{equation} k = \frac{\sqrt{2mE}}{\hbar}, \end{equation} and $k$ is imaginary, we must have $E \lt 0$. We now have four equations, which allows us to write \begin{equation} -\frac{\hbar^2 k}{m}=V\_0 a B, \end{equation} which implies that \begin{equation} \kappa = -\frac{maV\_0}{\hbar^2}, \end{equation} which implies that $V\_0 a$ must be negative to get a bound state and there is only one bound state. We can write the energy of this bound state as \begin{equation} E = -\frac{\hbar^2\kappa^2}{2m} = -\frac{ma^2V\_0^2}{2\hbar^2}, \end{equation} which lets us write the wavefunction \begin{align} \psi\_- &= Be^{\kappa x}\\ \psi\_+ &= -Ce^{-\kappa x}, \end{align} and so using the fact that $B=-C$, we get that \begin{equation} \psi(x) = Be^{-\kappa |x|}, \end{equation} and we could determine $B$ by normalization. #### Scattering States For these states, we can project onto the delta states which gives the idea of a forward and backward traveling state.
