A Universal Metric Space

Metric Space


Some common metric spaces are the normal Euclidean space with the metric $d(x,y)=|y-x|$. Another interesting metric is the discrete metric in which $d(x,y)=0$ if $x=y$ and $d(x,y)=1$ if $x\neq y$. Metric spaces can also be described as a space where we can define a suitable distance function in the sense that there must be a minimum distance between any two points in the space.

Universal Metric Spaces

It is known that there are many universal metric spaces in the sense that they contain an isometric embedding of every metric space of a certain type. In the case of Urysohn space, it contains an isometric property of every complete separable metric space.
