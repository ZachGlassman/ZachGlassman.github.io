Installation


Why we need installation

LaTeX can be run from the command line with a text editor and proper installed programs, however, I recommend using a LaTeX environment that simplifies matters a bit. Here we will give installation instructions for PC and Mac users as well as a new cloud based solution for LaTeX typesetting.

PC - MikTeX

  1. Go to MikTeX.org
  2. Download recommended installer.
  3. Install downlowded package.
  4. Ready to LaTeX!




Mac - MacTeX and TeXShop

  1. Go to http://www.tug.org/mactex/index.html
  2. On frontpage download and install latest package
  3. Go to http://pages.uoregon.edu/koch/texshop/
  4. Go to the obtaining section,download, and install
  5. Make sure that the Path setting under the "Engine" tab in TeXShop preferences is set to "usr/textbin," this should be done automatically. There is also a chance that your particular configuration requires a different configuration that is very system specific.
  6. Ready to LaTeX!


Cloud - ShareLaTeX

  1. Go to sharelatex.com
  2. Make an account
  3. Ready to LaTeX!
When you have chosen your particular enviornment, type the following code and attempt to compile it. 
\documentclass{article}
\begin{document}
This is my first \LaTeX document!  
\begin{equation}
     a^2+b^2=c^2
\end{equation}
I can write equations!
\end{document}
