An Introduction

What is LaTeX?

LaTeX is a document preparation system that produces high-quality typeset documents. It is a set of macro's over Donal Knuth's TeX typsetting language which effectively allows one actually write documents and leave design elements to the compiler (although these can be extensively customized as well). LaTeX is the preffered typesetting system for most academic fields and most people know it as a good way to typeset mathematical formulas. However, it can also can be used for posters, presentations, and many other useful applications. According the official guide, it can be pronounced either Lah-tech or Lay-tech.

What LaTeX is not.

LaTeX is not a word processor, and subsequently, not what you see is what you get (WYSWYG). This switches the focus from design to content.
