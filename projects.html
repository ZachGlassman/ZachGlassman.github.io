LaTeX Tutorial

This project was commissioned by the Claremont McKenna Department of Mathematics and is an introductory guide to the LaTeX typesetting language. It contains examples and instructions that should enable a student to typeset homework and other documents and furthermore, provide a general knowledge that simplify further exploration.

Go to LatexTutorial

Garduino

The Garduino is an automated gardening system that seeks to preserve water usage in places like Southern California. It was created for an electronics class project at Pomona College. It uses soil moisture sensors combined with an automated web scraper to make real-time determinations of the necessity for further watering. It is a featured Instructable and has over 68,000 page views.
